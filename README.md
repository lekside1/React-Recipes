# React-Recipes

A Recipe App written in React JS 

Choose your favorite Recipes 


How to run: 

Use npm or yarn 

    $ npm install or $ yarn install
    
Run scripts using npm or yarn 

    $ npm start
    $ npm dev 
    $ npm build 
    $ npm lint 
    $ npm test 
    $ npm test:watch 
    $ npm test:coverage 
    $ npm precommit
